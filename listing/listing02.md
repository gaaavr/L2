Что выведет программа? Объяснить вывод программы. 
Объяснить как работают defer’ы и их порядок вызовов.

```go
package main
import (
	"fmt"
)
func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}
func anotherTest() int {
	var x int
	defer func() {
		x++
		
	}()
	x = 1
	return x
}
func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Программа выведет в первом случае 2, во втором 1.
В test() x является именованным возвращаемым значением.
Поэтому при выходе из программы defer инкрементирует её
и получается 2. В второй же функции возвращаемое значение
не является именованным и в момент возврата оно фиксируется
в значении равном 1.
Defer`ы являются отложенными вызовами и срабатывают 
при выходе из программы. Они работают по принципу LIFO. 
Это означает, что последний объявленный defer будет 
выполнен первый (last in first out).
```
