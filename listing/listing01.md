Что выведет программа? Объяснить вывод программы.

```go
package main
import (
    "fmt"
)
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Код выведет слайс [77 78 79], который будет ссылаться на
массив a. Если поменять в массиве a значение элементов
с индексом от 1 до 4 ., то они также изменятся 
в слайсе b. Если добавлять элементы в слайс b, то исходные
элементы массива a будут меняться на вновь добавленные, пока
слайс не выйдет за пределы исходного массива. В этом случае
слайс начнёт ссылаться на новый массив, а исходный уже меняться
не будет.
```
